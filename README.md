# Secure Group Chat
[다운로드](https://github.com/hahaha98757/secure-group-chat/releases)

서버가 메시지를 읽을 수 없는 안전한 그룹 채팅 프로그램입니다.

## 사용법
1. 클라이언트를 실행하고, 서버에 접속합니다. (원래라면 서버는 24시간으로 공개되어 있어야 합니다. 그러나 소규모의 개인 프로젝트이므로 서버를 직접 실행해야 합니다.)
2. 닉네임을 입력합니다. 대소문자 구분 없이 영문, 숫자, 언더바(_)만 사용할 수 있으며, 중복된 닉네임은 사용할 수 없습니다.
3. 채팅방을 생성하거나 기존 채팅방에 입장하는 등 프로그램을 사용합니다.

## 명령어
명령어는 '/'로 시작하며, 대소문자를 구분합니다.

- /rename <name> - 닉네임을 변경합니다. 대소문자 구분 없이 영문, 숫자, 언더바(_)만 사용할 수 있습니다.
- /list \[/s | /u] - 세션 또는 유저 목록을 확인합니다. (서버 사용 가능)
- /create <name> \[password] - 세션을 생성합니다. (비밀번호는 선택 사항)
- /join <name> \[password] - 세션에 참가합니다. (비밀번호는 선택 사항)
- /request \[/a | /r] \[name] - 세션 참가 요청을 확인하거나 수락/거절합니다. (세션 주인 전용)
- /leave - 세션을 떠납니다.
- /kick <name> - 세션에서 유저를 추방합니다. (세션 주인 전용)
- /exit - 프로그램을 종료합니다. (서버 사용 가능)
- /send - 모든 클라이언트에게 메시지를 전송합니다. (서버 전용)
- /help - 도움말을 표시합니다. (서버 사용 가능)
